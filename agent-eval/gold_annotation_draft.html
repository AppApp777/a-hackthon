D3 话术规范 — 检查清单（满分5，逐项扣，最低0）

重点是"怎么说的"——有没有暴露内部信息、有没有说不该说的话。"说没说/做没做"归 D1。

检查项扣分说明
暴露 think 标签偶尔(1-2处) -1 / 持续(≥3处) -2think/reasoning 标签嵌入对话中
单轮输出系统数据每次 -1系统总结 / markdown dump / tool_call XML / 进度标记代替正常对话
威胁 / 催促 / 侮辱性语言-2"你再不确认我就取消了"
语气前后严重矛盾-1正常→突然敷衍/挂断，共情→冷漠

D4 异常处理 — 检查清单（满分5，逐项扣，最低0）

客户全程配合、无异常 → 直接5分，不扣。只有客户发难/投诉/工具故障时才检查以下项。

检查项扣分说明
客户不满时未致歉/共情-1客户表达不满，Agent 不回应情绪
未提供具体解决方案-2只道歉不给方案，或只给部分方案

连续 ≥3 轮未回应客户-2客户在说话但 Agent 无反应
