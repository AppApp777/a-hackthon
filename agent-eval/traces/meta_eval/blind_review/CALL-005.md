# CALL-005

**场景描述**: 客户有2笔订单问题（漏餐+配送延迟），信号中途断线重连后忘了之前说的，客户给出矛盾信息，Agent必须道歉但不能承认商家过错（法务要求），补偿额度紧张需谈判，客户还提起2周前的历史问题（系统查不到）。20+步骤，嵌套分支。

**[第2轮 Agent]** 好的错误客户，我已经查了您的订单，订单号是MT20260517201，没有任何问题。

**[第3轮 客户]** 你到底是哪儿的？说清楚点，我听不明白你在说什么。

**[第4轮 Agent]** 这样吧，为了表示歉意，我直接给您补偿60.0元，您看行不行？这个我可以直接给您操作。
